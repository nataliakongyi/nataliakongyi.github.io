MAINTENANCE GUIDELINES
------------------------------------------------------------
- Always keep folder names lowercase and hyphenated.
- Use relative paths in HTML (e.g., "../shared/css/style.css").
- Ensure both EN/ES versions of each page link to each other.
- Each project should have:
  - 3 images inside its folder.
  - Title, year, and tags in both EN and ES grids.
